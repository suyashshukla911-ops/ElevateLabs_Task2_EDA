# Task 2: Exploratory Data Analysis (EDA)

## Objective
To understand the dataset using statistics and visualizations.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Steps Performed

1. Generated summary statistics (mean, std, etc.)
2. Checked missing values
3. Created histograms for numeric features
4. Created boxplots to detect outliers
5. Built correlation matrix heatmap
6. Generated pairplot to analyze feature relationships

## Key Insights

- Age contains missing values.
- Fare distribution is right-skewed.
- Strong correlation observed between Pclass and Fare.
- Outliers present in Fare feature.

## Output
All visualizations saved as PNG files.
